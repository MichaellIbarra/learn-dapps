# Introducción a Token NFT
- Los tokens no fungibles (NFT) son activos digitales únicos que representan la propiedad de un artículo específico, como arte, música o coleccionables.

## Ventajas de los NFT
- Autenticidad: Los NFT permiten verificar la autenticidad y la propiedad de un activo digital.
- Escasez: Los NFT pueden ser creados en cantidades limitadas, lo que aumenta su valor percibido.
- Interoperabilidad: Los NFT pueden ser utilizados en diferentes plataformas y aplicaciones dentro del ecosistema blockchain.

## Tipos de NFT
- Arte digital: Obras de arte únicas que se venden como NFT.
- Coleccionables: Tarjetas, juguetes y otros artículos coleccionables representados como NFT.
- Música: Canciones y álbumes que se pueden comprar y vender como NFT.

## Token ERC-721
- El estándar ERC-721 es el más comúnmente utilizado para crear NFT en la blockchain de Ethereum.
- Proporciona una interfaz para la gestión de tokens no fungibles, permitiendo la transferencia y la propiedad de estos activos digitales únicos.

## Carácteristicas de ERC-721
- Unicidad: Cada token ERC-721 tiene un identificador único que lo distingue de otros tokens.
- Propiedad: Los tokens ERC-721 pueden ser propiedad de una sola dirección en la blockchain.
- Transferibilidad: Los tokens ERC-721 pueden ser transferidos entre direcciones, permitiendo la compra y venta de NFT.
- Metadata: Los tokens ERC-721 pueden incluir metadatos que describen el activo digital asociado.
- Aprobación: Los propietarios de tokens ERC-721 pueden aprobar a otros para gestionar sus tokens en su nombre.

## Diferencias entre ERC-20 y ERC-721
- ERC-20 es un estándar para tokens fungibles, mientras que ERC-721 es para tokens no fungibles.
- ERC-20 permite la intercambiabilidad entre tokens, mientras que cada token ERC-721 es único y no intercambiable.
- ERC-721 no son divisibles, mientras que los tokens ERC-20 pueden ser divididos en unidades más pequeñas.

## Términos comunes
- fungibilidad: La capacidad de un activo para ser intercambiado por otro del mismo tipo y valor.
